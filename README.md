# Projeto BHTRANS - Monitoramento de Frota

Este repositório contém o desenvolvimento de um sistema para rastrear os ônibus de Belo Horizonte em tempo real. A ideia surgiu da necessidade de verificar se os horários das linhas de BH estão sendo cumpridos de verdade, usando os dados que a prefeitura disponibiliza.

## 📌 O que o projeto faz:
A meta é bater os dados de GPS dos ônibus com o quadro de horários oficial. Atualmente, o foco é:
- Pegar a localização exata de cada veículo via API.
- Filtrar por número da linha (ex: 8207, 5106).
- Criar um histórico simples de atrasos por região.
